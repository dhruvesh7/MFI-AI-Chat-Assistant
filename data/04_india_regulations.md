# India Data Architecture and Regulatory Context

Money Forward operates within India's data regulation framework.

DEPA:
Data Empowerment and Protection Architecture.

Key Idea:
Individual controls their own data.

Account Aggregator Model:
- Consent-based sharing
- Encrypted data flow
- Data blind architecture
- No data storage by aggregator

Financial Inclusion:
Cash flow based lending for MSMEs.

Uses:
- GST invoices
- Bank transactions
- Digital footprint

DPDP Act 2024:

Rights:
- Access data
- Correct data
- Delete data

Penalty:
Up to 250 crore INR for data breach.

RBI Safeguards:
- Unauthorized transaction protection
- Fraud detection
- MuleHunter AI system
