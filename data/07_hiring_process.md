# Money Forward India Hiring Process

Selection Steps:

1. CV Application
2. Coding Test
3. Technical Interview
4. Managerial Interview
5. HR Discussion

Coding Test:
- Online assignment
- Data structures
- Problem solving
- Coding questions

Technical Interview:
- Programming basics
- Backend concepts
- APIs
- Database
- Problem solving

Managerial Round:
- Team fit
- Communication
- Ownership

HR Round:
- Salary discussion
- Internship duration
- Joining timeline
